# Problemas conocidos: compilación de Windows 10 Team 2020 Update Preview 

Esta página proporciona una lista de problemas conocidos para la actualización de la versión preliminar de Windows 10 Team 2020. Los problemas que se han corregido desde el lanzamiento de la versión preliminar se enumeran en la tabla 2. 

#### Tabla 1. Problemas conocidos: compilación de Windows 10 Team 2020 Update Preview 

| Problema                                                                     | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Problemas de interfaz de usuario, audio y gráficos**                            | Puede experimentar diversos problemas de la interfaz de usuario y los gráficos si instala la compilación de vista previa, pero no instale inmediatamente las actualizaciones de firmware necesarias más adelante. Microsoft planea resolver estos problemas en la versión de compilación de Windows 10 Team 2020 Update.                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **El inicio de sesión se ve afectado cuando se aplican las directivas de acceso condicional**      | -Se produce un error en el inicio de sesión al intentar iniciar sesión en aplicaciones como Microsoft whiteboard. Los usuarios deben iniciar sesión en [mis reuniones y archivos](https://support.microsoft.com/help/4506480/sign-in-to-see-your-meetings-and-files-on-surface-hub) desde el menú Inicio.<br>-Se produce un error en el inicio de sesión si su cuenta de usuario está registrada en otro inquilino de Azure AD que el usado por Surface hub para unirse a Azure AD.<br><br>Microsoft planea resolver estos problemas en la versión de compilación de Windows 10 Team 2020 Update.                                                                                                                                                                                              |
| **Windows Update solicita la instalación de nuevos drivers cuando no hay ninguno disponible** | Cuando vaya a **configuración**  >  **Actualizar & seguridad**de  >  **Windows Update** después de instalar la actualización 2020 de Windows 10 y las actualizaciones de firmware necesarias, es posible que vea el siguiente error:<br><br>-"Es posible que un controlador actual de tu PC sea mejor que el controlador que estamos tratando de instalar. Seguiremos intentando instalarlo.<br><br>Este error puede ignorarse de forma segura. Microsoft está investigando de forma activa este problema.                                                                                                                                                                                                                                                                              |
| **Pregunta para quitar la unidad USB antes de tiempo durante la primera ejecución**               | Al usar un paquete de aprovisionamiento para configurar Surface Hub durante la primera ejecución, se le pedirá que quite la unidad USB antes de que el dispositivo pueda leer el archivo de configuración de Surface Hub. Si está usando un archivo para configurar su cuenta de dispositivo, ignore el mensaje. Microsoft está investigando de forma activa este problema.                                                                                                                                                                                                                                                                                                                                                                          |
| **Se muestra un error al descargar aplicaciones desde Microsoft Store**          | Para descargar una aplicación de Microsoft Store, verá un mensaje que le pedirá que inicie sesión. Un problema conocido hace que aparezca un error durante el inicio de sesión, pero esto no afecta a su capacidad de descargar aplicaciones. Microsoft está investigando de forma activa este problema.                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Surface Hub 2S pierde intermitentemente la conexión WiFi**                  | Desconecta el dispositivo y vuelve a conectarlo, o usa un cable de Ethernet para conectarte a Internet. Microsoft está investigando de forma activa este problema.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| **Surface Hub 2S no se puede reanudar de forma intermitente desde la suspensión**              | Surface Hub 2S es posible que no se reinicie por error de forma intermitente y parezca que deja de responder en una pantalla que muestra el logotipo de Microsoft. Intenta desconectar el dispositivo y volver a conectarlo.<br><br>**Para evitar este problema:**<br><br>1. en Surface Hub, Abra **configuración** y escriba sus credenciales de administrador cuando se le solicite.<br>2. navegue a la sesión de **Surface Hub**  >  **& energía**.<br>3. en **cuando el dispositivo vaya a suspender, use esta configuración de energía**y seleccione **conectado al modo de espera.**<br>4. en **cuando no hay nadie presente, ponga el dispositivo en suspensión después**de, seleccione **nunca.**<br><br>Microsoft planea resolver este problema en una actualización de firmware antes de la versión final de la actualización 2020 del equipo de Windows 10. |
